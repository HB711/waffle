# 5D Experiments

This folder contains the 5D research branch of the project.

## Suggested Entry Points

- `5dofv4.py`: tempered multi-state workflow.
- `314.py`: complete multi-state experiment with adaptive overlap penalties.
- `sequence.py`: staged training flow.
- `local.py`: local-box sampling around a selected minimum.
- `GS.py`: parameter sweep and plotting workflow.

## Folder Philosophy

The 5D branch is still exploratory. Several scripts represent nearby design alternatives rather than a single stable implementation. The current cleanup groups them together and leaves naming and history intact so experimental context is not lost.
